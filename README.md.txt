# Staff Flow - HR Management System

Staff Flow is a modern Human Resource Management System built to simplify employee management, attendance tracking, leave management, notifications, and reporting.

## Features

- Employee Authentication
- Role-based Access (Admin & Employee)
- Attendance Management
- Leave Management
- Notifications
- Employee Profiles
- Dashboard Analytics
- Admin Management Panel
- Automatic Leave Balance Calculation
- Check In / Check Out
- Return to Work
- On Leave Status

## Technologies Used

### Frontend
- React
- Vite
- Axios
- CSS Modules
- Chart.js

### Backend
- Node.js
- Express.js
- MySQL
- JWT Authentication
- Multer

## Project Structure

```
staff-flow
│
├── staff-flow-backend
└── staff-flow-frontend
```

## Installation

### Backend

```bash
cd staff-flow-backend
npm install
npm start
```

### Frontend

```bash
cd staff-flow-frontend
npm install
npm run dev
```
